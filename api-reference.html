        Anoma.Node.Router.Addr

        An address to which we can send a message. The server, if known, is a local actor which can receive it directly; otherwise, the mssage will be sent via the central router. If the server is known, but the id is not, then this is a local-only engine, which can only talk to other local engines. (Hence, at least one of id and server must be known; potentially both are.)
